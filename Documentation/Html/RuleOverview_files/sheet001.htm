Rule overview  
Ruletype Description
ConstantValueRule The value of an attribute is constant over more then one sequence item. The last item in the sequence should not be present.
CriticalAttributeRule Integration-critical attributes that need to be equal (copied or generated locally).
DoseCommentRule Shall be present if Referenced RT Plan Sequence is NOT PRESENT If present it should have the same value as RT Plan Description and it should not be empty
EqualToRule The attribute shall have the same value as another attribute
GridFrameOffsetVectorRule Value should be 0 of first index
Check that every index has a higher value than the index before (start this at second index ofcourse)
HasValueIfPresentRule Checks if attribute exists it must have a value
HighBitRule Value = Bits Stored(0028,0101) - 1
ImageOrientationPatientRule if (ValueRule for Image Type(0008,0008) with value == AXIAL)
then values are (+/-1, 0, 0, 0, +/1, 0) with AR5: ToleranceRule for every value, tolerance of 0.001
InteroperabilityRule Checks if an attribute has the same value between datasets of different commands.
For example: between C-FIND-RSP and N-ACTION-RQ.
The TMS actor is typically the actor which checks the interoperabilities.
InterrelationRule Checks if an attribute has the same value between the request and response dataset of the same command.
The TDD actor is typically the actor which checks an attribute of the response dataset with the request send earlier.
ManufacturerMappingRule IHE requires that this element be present, and should contain the manufacturer of the equipment creating the structure set, plan, or dose. 
If the equipment is storing and forwarding information, the value of this element shall be preserved.
 If a new plan is created from a previous plan, the manufacturer of the equipment producing the new plan shall insert their identifier in this element.
If a new structure set is created from a previous structure set, the manufacturer of the equipment producing the new structure set shall insert their identifier in this element.
If an application resamples and re-exports a series of CT images, or modifies an instance then this element must be present, and must contain the model name of the equipment doing the resampling.
MappingRule Checks if an attribute in a request dataset has the same value as defined in the
response dataset.
NotPresentRule Checks if the attribute or sequence is NOT present
NotEqualToRule Checks if the attribute is not equal to the given value
NrIntOfSequenceItemsRule Checks that a sequence has the same number of sequence items indicated by the given value.
OptionalRule Reports if an attribute is present or not.
PixelSpacingRule For CTImage IOD, non-isotropic pixels are outside the scope of the profile.
For RT Dose IOD, pixel spacing may be nonisotropic.
TissueHeterogeneityCorrectionRule Checks that the atrribute value is equal as in referencedata
ValueInRangeRule Checks if the attribute is in the given range.
ValueListRule Checks if the attribute has the right value given in the specified list of values.
ValueRule Checks if the attribute has the right value.
