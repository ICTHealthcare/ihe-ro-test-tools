Test Scenario 1 Geometric Planner Equidistant
Instructions for use test scenarios
Module Overview
Description:
This test scenario provides the geometric planner with RO-1 and RO-7 and receives RO-3
- Checks the correct storage of Single/contoured CT series dataset (DICOM assoc. & interaction level)
- Checks the correct storage of the Structure set  (DICOM association & interaction level)
- Validation of the created Geometric Plan (RO-3) (including RO critical modules)
- Verifies the critical attribute mapping requirements
Transactions sent to actor:
RO-1 (Single/Contoured Series Image Retrieval)
RO-7 (Structure Set Retrieval)
Expected transactions from actor:
RO-3 (Geometric Plan Storage)
Used scenario dataset:
This scenario uses the equidistant reference test dataset.
