Test Scenario 2 Contourer actor
Instructions for use test scenarios
Module Overview
Description:
This test scenario provides the contourer actor with RO-1 and receives RO-2 or RO-BRTO-II-1
Dialog as shown below is shown when this scenario starts.
Choose what kind of structure set is expected to receive.
After that this dialog will be shown.
Select the directory where the Image Serie is located, these images will be used by RO-1.
Select the directory where the test tool must store all received objects.
Validation:
- Checks the correct storage of Single Series CT dataset (DICOM association & interaction level)
- Validation of the created Structure set RO-2 or RO-BRTO-II-1 (including RO critical modules)
- Verifies the critical attribute mapping requirements
Transactions sent to actor:
RO-1 (Single/Contoured Series Image Retrieval)
Expected transactions from actor:
RO-2 (Structure Set Storage) OR    
RO-BRTO-II-1 (High-resolution Structure Set Storage)
Used scenario dataset:
This scenario uses the test dataset specified by the user.
