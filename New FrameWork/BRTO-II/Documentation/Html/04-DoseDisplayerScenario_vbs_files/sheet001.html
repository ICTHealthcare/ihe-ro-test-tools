Test Scenario Dose Displayer
Instructions for use test scenarios
Description:
This test scenario provides the dose displayer with RO-1, RO-7 or RO-BRTO-II-2, RO-9  and RO-10
Dialog as shown below is shown when this scenario starts.
Choose the kind of structure set that will be used by this scenario.
After that this dialog will be shown.
Select the directory where the Image Series is located, these images will be used by RO-1.
Select the RT Structure Set file, this fill will be used by RO-7 or RO-BRTO-II-2.
Select the RT Plan file, this fill will be used by RO-9.
Select the RT Dose file, this fill will be used by RO-10.
Validation:
- Checks the correct storage of Single/contoured CT series dataset (DICOM assoc. & interaction level)
- Checks the correct storage of the Structure set  (DICOM association & interaction level)
- Checks the correct storage of the (Dosimetric) Plan (DICOM association & interaction level)
- Checks the correct storage of the RT Dose (DICOM association & interaction level)
   
   
Transactions sent to actor:
RO-1 (Single/Contoured Series Image Retrieval)
RO-7 (Structure Set Retrieval) OR      
RO-BRTO-II-2 (High-resolution Structure Set Retrieval)  
RO-9 (Dosimetric Retrieval)
RO-10 (Dose Retrieval)
Expected transactions from actor:
n.a.
Used scenario dataset:
This scenario uses the test dataset specified by the user.
