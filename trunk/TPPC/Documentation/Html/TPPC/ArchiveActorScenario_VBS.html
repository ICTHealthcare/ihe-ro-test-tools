Archive Actor Scenario
Instructions for use test scenarios
Description:
1. Before starting the scenario, make sure the Archive actor is active and listening for data.
2. Start the scenario.
3. A pop-up appears in which the user can select which kind RTPlans are to be send by the scenario. The user can choose the following: 
·          Basic Static Beam Storage (TPPC-01)
·        Basic Static MLC Beam Storage (TPPC-03)
·        Arc Beam Storage (TPPC-05)
·        MLC Arc Beam Storage (TPPC-07)
·        Conformal Arc Beam Storage (TPPC-09)
·        Hard Wedge Beam Storage (TPPC-11)
·        Virtual Wedge Beam Storage (TPPC-13)
·        Motorized Wedge Beam Storage (TPPC-15)
·        Static Electron Beam Storage (TPPC-17)
·        Step & Shoot Beam Storage (TPPC-19)
·        Sliding Window Beam Storage (TPPC-21)
·        IMAT/VMAT Beam Storage (TPPC-23)
·        Photon Applicator Beam Storage (TPPC-29)
·        Photon Applicator Arc Beam Storage (TPPC-31)
4. After selecting the desired transactions click on OK and the scenario sends the RTPlans. The test tools now start to listen.
5. Send the opposite transactions with the archive actor to the test tools. The following transactions are possible depending on the send transaction by DVTk.
·        Basic Static Beam Retrieval (TPPC-02)
·        Basic Static MLC Beam Retrieval (TPPC-04)
·        Arc Beam Retrieval (TPPC-06)
·        MLC Arc Beam Retrieval (TPPC-08)
·        Conformal Arc Beam Retrieval (TPPC-10)
·        Hard Wedge Beam Retrieval (TPPC-12)
·        Virtual Wedge Beam Retrieval (TPPC-14)
·        Motorized Wedge Beam Retrieval (TPPC-16)
·        Static Electron Beam Retrieval (TPPC-18)
·        Step & Shoot Beam Retrieval (TPPC-20)
·        Sliding Window Beam Retrieval (TPPC-22)
·        IMAT/VMAT Beam Retrieval (TPPC-24)
·        Photon Applicator Beam Retrieval (TPPC-30)
·        Photon Applicator Arc Beam Retrieval (TPPC-32)
6. Click stop listening start validating. Validation of the received transactions starts.
