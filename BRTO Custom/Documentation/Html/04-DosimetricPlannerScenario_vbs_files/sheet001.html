Test Scenario Dosimetric Planner
Instructions for use test scenarios
Module Overview
Description:
This test scenario provides the dosimetric planner with RO-1, RO-7 and RO-8 and receives RO-4 and RO-5
Dialog as shown below is shown when this scenario starts.
Select the directory where the Image Series is located, these images will be used by RO-1.
Select the RT Structure Set file, this fill will be used by RO-7.
Select the RT Plan file, this fill will be used by RO-8 (Optional).
Select the directory where the test tool must store all received objects.
Validation:
- Checks the correct storage of Single/contoured CT series dataset (DICOM assoc. & interaction level)
- Checks the correct storage of the Structure set (DICOM association & interaction level)
- Checks the correct storage of the (Geometric) Plan (DICOM association & interaction level)
- Validation of the created Dosimetric Plan (RO-4) (including RO critical modules)
- Validation of the created Dose (RO-5) (including RO critical modules)
- Verifies the critical attribute mapping requirements
Transactions sent to actor:
RO-1 (Single/Contoured Series Image Retrieval)
RO-7 (Structure Set Retrieval)
RO-8 (Geometric Plan Retrieval) Optional
Expected transactions from actor:
RO-4 (Dosimetric Plan Storage)
RO-5 (Dose Storage)
Used scenario dataset:
This scenario uses the test dataset specified by the user.
