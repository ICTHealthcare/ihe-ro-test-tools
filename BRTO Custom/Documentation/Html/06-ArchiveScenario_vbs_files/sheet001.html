Test Scenario Archive actor
Instructions for use test scenarios
Description:
This test scenario provides the archive actor with all data objects and receives the data objects.
Dialog as shown below is shown when this scenario starts.
Select the directory where the Image Series is located, these images will be used by the validation of RO-1.
Select the RT Structure Set file, this file will be used by the validation of RO-7.
Select the RT Plan (geometric) file, this file will be used by the validation of RO-8.
Select the RT Plan (dosimetric) file, this file will be used by the validation of RO-9.
Select the RT Dose file, this file will be used by the validation of RO-10.
Validation:
- Storage of the reference test data objects to the archive
- Retrieval of the reference test data object from the archive
- Comparison of the test data objects received, with the original reference data.
- Checks ensuring level 2 SCP conformance.
Transactions sent to actor:
RO-2 (Structure Set Storage)
RO-3 (Geometric Plan Storage)
RO-4 (Dosimetric Plan Storage)
RO-5 (Dose Storage)
RO-11 (Resampled/Combined CT Series Storage)
Expected transactions from actor:
RO-1 (Single/Contoured Series Image Retrieval)
RO-7 (Structure Set Retrieval)
RO-8 (Geometric Plan Retrieve)
RO-9 (Dosimetric Plan Retrieve)
RO-10 (Dose Retrieve)
Used scenario dataset:
This scenario uses the test dataset specified by the user.
