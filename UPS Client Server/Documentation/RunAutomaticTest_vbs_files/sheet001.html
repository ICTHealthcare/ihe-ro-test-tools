Run automatic test
Description:
With this test scenario a automatic test can be executed.
All made automatic tests have a xml file with the name of the automatic test and are located in the "<Instalation dir>\Sessionfiles\Scripts" directory.
In the "<Instalation dir>\Sessionfiles\Scripts\CurrentAutomaticTestingConfiguration.xml" configuration file in the CurrentAutomaticTestName node fill in the path of the automatic test xml file. 
This test will then be executed by starting this scenario.
